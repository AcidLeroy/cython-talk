Cython

Using real-time code in CPython

Created by Cody W. Eilar

Motivation

Single Definition Rule

If you have a great C/C++ algorithm, why implement it again in Python?

Testing real-time code against experimental code

Create unit tests in python to verify that both codes have the same output

Rapid prototyping of real time code

Use the wealth of algorithms created in real-time code to build full fledged applications in hours

Speed up Python code

Have some functions that could use some speed-up? Compile them with Cython to get orders of magnitude in improvement!

What is Cython?

Cython is an

Optimizing static compiler for both the Python and the extended Cython programming language.

They Claim...

That it "makes writing C extensions for Python as easy as Python itself". Experience has shown this is not the same for C++.
